## 🎨 Design Features

### 1. **Glassmorphism Effects**
- Frosted glass cards with blur backdrop
- Semi-transparent surfaces
- Layered depth with shadows
- Premium feel reminiscent of iOS and modern UI

### 2. **Neon Cyberpunk Aesthetics**
- Electric cyan (#00d9ff) primary color
- Magenta (#ff006e) secondary accents
- Purple (#7b2cbf) tertiary highlights
- Glowing borders and hover effects
- Dark background (#0a0e1a) for contrast

### 3. **Animated Grid Background**
- Subtle moving grid pattern
- Cyberpunk/technical aesthetic
- Creates sense of depth and motion
- Doesn't distract from content

### 4. **Floating Gradient Orbs**
- Animated background effects
- Soft glowing orbs that float
- Creates atmospheric depth
- Smooth parallax-like motion

### 5. **Futuristic Typography**
- **Orbitron**: Bold, geometric display font for titles
- **Noto Sans JP**: Japanese-inspired body text
- **JetBrains Mono**: Technical monospace for labels
- Uppercase headings with gradient text effects

### 6. **Holographic Gradients**
- Multi-color gradient effects
- Animated gradient shifts
- Card hover states reveal subtle rainbow effects
- Text gradients on all major headings

### 7. **Interactive Elements**
- Smooth animations on scroll
- Hover effects with glow and lift
- Staggered fade-in animations
- Micro-interactions on all clickable elements

### 8. **Floating Cards**
- Hero section has animated floating cards
- Glassmorphic background with blur
- Continuous floating animation
- Interactive hover states

## 🎨 Customization Options

Want to adjust the aesthetic? Easy changes:

**Make it more neon/bright:**
- Increase glow opacity in CSS variables
- Add more gradient variety
- Brighter accent colors

**Make it darker/more subtle:**
- Reduce glow effects
- Use single-color scheme
- Simplify gradients

**Change color theme:**
- Update CSS color variables
- Consistent across entire site
- Takes 2 minutes
